# CSV to PDF Generator

Проект на Laravel 10 и PHP 8 для генерации PDF файлов из CSV данных с использованием PDF шаблонов.

## Функционал

1. **Загрузка файлов**: Загрузка CSV файла с данными и PDF шаблона
2. **Сопоставление полей**: Автоматическое или ручное сопоставление полей из CSV с переменными в PDF
3. **Генерация PDF**: Создание объединенного PDF файла со всеми записями из CSV

## Требования

- PHP >= 8.0
- Composer
- Расширения PHP: mbstring, xml, gd, zip

## Установка

1. Установите зависимости:
```bash
composer install
```

2. Скопируйте файл `.env.example` в `.env`:
```bash
cp .env.example .env
```

3. Сгенерируйте ключ приложения:
```bash
php artisan key:generate
```

4. Настройте права доступа:
```bash
chmod -R 775 storage bootstrap/cache
```

5. Запустите сервер разработки:
```bash
php artisan serve
```

Откройте браузер и перейдите по адресу: `http://localhost:8000`

## Использование

1. Загрузите CSV файл с данными (первая строка должна содержать заголовки)
2. Загрузите PDF шаблон (может содержать переменные в формате `{{variable}}`, `{variable}`, `[variable]` или `$variable`)
3. Сопоставьте поля из CSV с переменными в PDF
4. Нажмите "Сгенерировать PDF" для создания объединенного файла

## Структура проекта

- `app/Http/Controllers/PdfGeneratorController.php` - Контроллер для обработки запросов
- `app/Services/CsvParserService.php` - Сервис для парсинга CSV файлов
- `app/Services/PdfService.php` - Сервис для работы с PDF (извлечение полей и генерация)
- `resources/views/index.blade.php` - Главная страница с интерфейсом

## Зависимости

- **Laravel Framework** - веб-фреймворк
- **TCPDF** - библиотека для работы с PDF
- **FPDI** - библиотека для импорта страниц из существующих PDF

## Примечания

- CSV файлы должны иметь заголовки в первой строке
- PDF шаблон должен содержать переменные в одном из поддерживаемых форматов
- Сгенерированные PDF файлы сохраняются в `storage/app/generated/`
